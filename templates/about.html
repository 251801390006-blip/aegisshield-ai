About AegisShield AI

AegisShield AI is an advanced cybersecurity platform built with Python, Flask, and Scikit-Learn to demonstrate real-world AI/ML applications in cyber threat detection.

🎯 Platform Purpose

Built as an internship showcase project to demonstrate proficiency in full-stack development, machine learning integration, cybersecurity concepts, and professional UI/UX design. Suitable for placement portfolios and technical demonstrations.

🤖 AI/ML Stack

  • Naive Bayes + Logistic Regression (Spam)
  • Gradient Boosting + 20-feature engineering (Phishing)
  • Shannon entropy + heuristic rules (Password)
  • Static analysis + magic bytes (Malware)

🔧 Technical Stack

  • Python 3.11 + Flask 3.x (Backend)
  • SQLAlchemy + SQLite/PostgreSQL (DB)
  • Bootstrap 5 + Chart.js (Frontend)
  • ReportLab (PDF Generation)
  • Gunicorn + Render (Deployment)

🔒 Security Features

  • CSRF Protection (Flask-WTF)
  • Password Hashing (Werkzeug bcrypt)
  • Rate Limiting (Flask-Limiter)
  • XSS Protection (Bleach sanitization)
  • Secure HTTP headers
